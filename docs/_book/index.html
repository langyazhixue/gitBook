Introduction

云霁科技大前端

以该文档作为云霁科技前端的代码规范、交互规范、Demo、Issue，分享等管理的中心，如果对前端相关有组件需求，交互优化需求请在该项目的 gitlab 提 issue，同时希望大家能共同参与，维护好

Contribute 方式

  1. 安装编译文档 gitbook
$ npm install -g gitbook-cli --registry=https://registry.npm.taobao.org
$ gitbook fetch
$ git clone <repo-地址>  && cd <repo-dir>
$ gitbook build
  1. 修改文档，打分支，通过 gitlab 提交 MR
