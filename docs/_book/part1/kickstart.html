项目安装

项目安装

安装全局依赖

  1. node@latest
  2. npm install -g yarn eslint

clone 项目

$ git clone <project-repo-url>
$ cd <project>
$ yarn

启动项目

$ npm run start

打开 http://localhost:3000 即可运行

编译发布

$ npm run build

配置 Proxy

为了能够实现直接调用后端 api，可以在项目中配置到后端的 proxy， 可配置 proxy 的地方有两个：

  1. proxy.json 全局配置
  2. .proxy 本地配置

.proxy 中的配置优先，可以覆盖 proxy.json 的配置

基本的配置方式为

  • api 对应后端 server
  • endpoints 对应需要代理的路由，可以使用 * 来匹配
{
  "default": {
    "api": "http://cloudres.idcos.net",
    "endpoints": [
      "/api/*",
      "/dcosapp/**/",
      "/catalog/**/",
      "/uam/*"
    ]
  }
}
